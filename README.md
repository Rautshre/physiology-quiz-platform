# Physiology MCQ Practice Platform

A simple frontend-based website for practicing **Physiology MCQs** with more than **1300+ questions** organized for efficient learning and revision.

## Features

- ✅ 1300+ Physiology MCQs
- ✅ Topic-wise question practice
- ✅ Difficulty level filtering
- ✅ Random test generation
- ✅ Topic-wise test creation
- ✅ User-friendly interface for quick revision

## Functionalities

### Topic-wise Practice
Users can select specific physiology topics and practice questions accordingly.

### Difficulty-based Sorting
Questions are categorized by difficulty level to improve learning progressively.

### Random Test Generation
Generate randomized tests for quick self-assessment.

### Topic-wise Tests
Practice tests based on selected physiology topics.

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Goal

The aim of this platform is to make physiology MCQ practice easier, more structured, and effective for students preparing for exams and revisions.
